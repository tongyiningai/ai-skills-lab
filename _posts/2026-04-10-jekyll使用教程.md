## 写在前面

本教程面向刚刚接触 Jekyll 的初学者，以本站（ai-skills-lab）为例，带你了解 Jekyll 的基本用法，并成功发布一篇博客文章。

Jekyll 是一个静态网站生成器，它把 Markdown、Liquid 模板等文件转换成完整的 HTML 网站，特别适合搭建博客或技术文档站。

## 1. Jekyll 环境搭建

### 安装 Ruby

Jekyll 基于 Ruby 开发，首先需要安装 Ruby（建议版本 3.x）。  
Windows 用户可安装 [RubyInstaller](https://rubyinstaller.org/)，macOS/Linux 用户使用系统包管理器即可。

### 安装 Jekyll 和 Bundler

打开终端，执行：

bash
```
gem install jekyll bundler
```
